Modules in Javascript

There are two types of Modules
1.Import Modules
2.Export Modules


Import Modules

Import Module is used to import from one file to another file


Export Module

Export Module consist of two types

1.Named Export
2.Default Export


1)Named Export

Named export are used to export several values.During the import,it is mandatory to same name of the corresponding object.

2)Default Export

Default Export are useful to export a single object,function,variable,During the import,we can use name to the import
